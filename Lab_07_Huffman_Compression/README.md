# Оптимизация хранения данных: Алгоритм Хаффмана (C++)

Проект демонстрирует реализацию классического алгоритма сжатия данных без потерь на основе префиксных кодов Хаффмана. В качестве набора данных для сжатия используются характеристики из открытого набора NSynth.

## Особенности реализации

* **Построение дерева кодирования:** Для формирования оптимального префиксного дерева применяется очередь с приоритетом (двоичная куча) с собственным алгоритмом сравнения узлов. Это обеспечивает высокую скорость вычисления кодов.
* **Побитовая упаковка:** Сжатая последовательность не просто хранится в оперативной памяти в виде текста, а упаковывается на уровне битов в полноценные байты и сохраняется в бинарный файл вместе с таблицей частот. Это позволяет получить реальное физическое сокращение объема файла.
* **Оценка эффективности:** Встроенный модуль тестирования автоматически генерирует выборки различного объема, проводит сжатие и вычисляет итоговый коэффициент сжатия.
* **Собственный парсер:** Реализован легковесный метод чтения и извлечения целевых параметров из файлов формата JSON напрямую через потоки, без использования сторонних библиотек.

## Технические детали

* Использование хэш-таблиц для эффективного подсчета частот символов за линейное время.
* Безопасная работа с бинарными файлами.
* Строгое управление памятью: реализована рекурсивная очистка узлов дерева для предотвращения утечек оперативной памяти при множественных вызовах алгоритма.
