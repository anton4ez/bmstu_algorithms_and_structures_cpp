# Алгоритмы и структуры данных (C++)

Портфолио проектов, выполненных в рамках дисциплины "Типы и структуры данных" в МГТУ им. Н.Э. Баумана.

## О репозитории
Данный репозиторий содержит реализации сложных алгоритмов, вычислительных структур данных и парсеров. Основной упор сделан на оптимизацию использования оперативной памяти, математическое профилирование времени выполнения и применение принципов безопасного управления ресурсами.

## Стек технологий
* **Язык:** C++
* **Инструментарий:** Стандартная библиотека шаблонов
* **Архитектурные подходы:** Оптимизация локальности кэша процессора, управление асимптотической сложностью алгоритмов, побитовая арифметика, разработка конечных автоматов.

## Каталог проектов

| Проект | Описание | Ключевые концепции |
| :--- | :--- | :--- |
| **[Лабораторная 1: Матрицы](Lab_01_Matrix_Multiplication/)** | Бенчмаркинг алгоритмов умножения плотных матриц. | Алгоритм Винограда, устранение кэш-промахов, оценка сложности $O(N^3)$. |
| **[Лабораторная 2: Профилирование](Lab_02_Recursion_vs_Iteration/)** | Оценка накладных расходов процессорного времени. | Сравнение рекурсии и итерации, анализ выделения памяти в стеке вызовов. |
| **[Лабораторная 3: Длинная арифметика](Lab_03_BigInt_Fibonacci/)** | Структура для работы с числами произвольной длины. | Фибоначчиева система счисления (база на теореме Цекендорфа). |
| **[Лабораторная 4: Разреженные матрицы](Lab_04_Sparse_Matrices/)** | Оптимизация памяти при работе с разреженными графами. | Кольцевая схема Рейнбольдта-Местеньи, транспонирование in-place за $O(1)$. |
| **[Лабораторная 5: Интерпретатор](Lab_05_Expression_Evaluator/)** | Парсинг и вычисление арифметико-логических выражений. | Алгоритм сортировочной станции (Shunting-yard), стековая вычислительная машина. |
| **[Лабораторная 7: Сжатие данных](Lab_07_Huffman_Compression/)** | Архиватор данных с побитовой упаковкой в бинарный файл. | Алгоритм Хаффмана, очередь с приоритетами (Min-Heap). |
| **[Лабораторная 8: Задача коммивояжёра](Lab_08_TSP_Ant_Colony/)** | Поиск оптимального маршрута (NP-трудная задача). | Рекурсивный поиск в глубину (DFS), Муравьиный эвристический алгоритм. |
| **[Лабораторная 9: WebAssembly Parser](Lab_09_WASM_Binary_Parser/)** | Низкоуровневый разбор бинарного формата исполняемого кода. | Конечные автоматы (FSM), декодирование LEB-128 и стандарта IEEE-754. |
