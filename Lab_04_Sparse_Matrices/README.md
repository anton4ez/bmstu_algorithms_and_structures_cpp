# Разреженные матрицы и оптимизация памяти (C++)

Проект посвящен алгоритмам сжатия и обработки матриц, состоящих преимущественно из нулей. Подобные структуры данных часто применяются в машинном обучении и анализе данных для экономии оперативной памяти и ускорения вычислений.

## Особенности реализации

В основе проекта лежит кольцевая схема Рейнбольдта-Местеньи. Ненулевые элементы и их координаты хранятся в виде системы закольцованных списков в непрерывных одномерных массивах. Это позволяет избежать создания избыточных структур и ускоряет доступ к данным со стороны процессора.

### Ключевые алгоритмы

* **Сжатие за один проход:** Формирование списков происходит в процессе чтения данных без выделения промежуточной памяти, что сводит издержки к абсолютному минимуму.
* **Мгновенное транспонирование:** За счет кольцевой природы указателей, транспонирование матрицы выполняется путем простой подмены ссылок (строки меняются со столбцами). Сами элементы не перемещаются в памяти, что дает вычислительную сложность O(1).
* **Быстрая распаковка:** Восстановление исходной геометрии матрицы происходит путем прямого обхода построенных связей, минуя ресурсоемкие вложенные циклы.

## Технические детали
* Использование стандартных контейнеров для безопасного управления памятью.
* Обработка исключений для защиты от некорректных структур данных при чтении из текстовых файлов.
