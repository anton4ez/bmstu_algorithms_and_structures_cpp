# Интерпретатор арифметических и побитовых выражений (C++)

Проект представляет собой реализацию калькулятора, преобразующего выражения из инфиксной нотации в постфиксную (обратную польскую запись) с последующим вычислением. Поддерживаются операции над десятичными и двоичными числами (до 8 бит).

## Реализованные возможности

* **Лексический анализ:** Разбор входной строки на операнды, операторы и скобки.
* **Алгоритм сортировочной станции:** Трансляция выражений с учетом приоритетов операторов и вложенности скобок.
* **Стековая машина:** Вычисление итогового значения выражения на основе постфиксной записи.

### Поддерживаемые операции
1. **Побитовые:** Отрицание (`~`), двоичные сдвиги влево (`<<`) и вправо (`>>`).
2. **Логические сравнения:** Равно (`=`), не равно (`!=`), больше/меньше (`>`, `<`, `>=`, `<=`).
3. **Системы счисления:** Автоматическое распознавание и конвертация между десятичной и двоичной системами.

## Особенности реализации
* **Собственная структура данных:** Реализован шаблонный класс `Stack` на базе динамического массива.
* **Безопасность:** Многоуровневая обработка исключений при некорректном вводе (несогласованные скобки, неизвестные токены, недостаток операндов).
* **Принципы RAII:** Автоматическое управление памятью через контейнеры стандартной библиотеки.
