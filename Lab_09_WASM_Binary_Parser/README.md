# WebAssembly Binary Parser & Finite State Machine (C++)

Данный проект представляет собой низкоуровневый парсер бинарного формата WebAssembly (WASM). Инструмент осуществляет разбор структуры файла, валидацию заголовков и глубокий анализ секции исполняемого кода (Section ID = 10) на основе теории конечных автоматов.

## Архитектура и возможности

* **Парсинг бинарных структур:** Безопасное побайтовое чтение бинарного файла напрямую в оперативную память.
* **Декодирование LEB-128:** Реализованы алгоритмы распаковки целых чисел переменной длины (Little Endian Base 128), используемых в спецификации WASM для типов `u32`, `u64` и `s32`.
* **IEEE-754 поддержка:** Декодирование чисел с плавающей точкой (`f32`, `f64`) из сырых байтов.
* **Стековая машина состояний:** Разбор потока инструкций осуществляется с помощью конечного автомата, который корректно отслеживает вложенность блоков (`block`, `loop`, `if-else`) и управляет отступами абстрактного синтаксического дерева.

## Модульное декодирование инструкций
Архитектура позволяет селективно декодировать определенные группы опкодов виртуальной машины WebAssembly:
1. `Control Instructions` (ветвления, вызовы функций, управление стеком блоков).
2. `Numeric Instructions` (арифметика, логика и работа с константами `i32/i64/f32/f64`).
3. `Memory Instructions` (чтение/запись в линейную память, управление размером).
4. `Table Instructions` (работа с косвенными вызовами).

## Технический стек
* **Язык:** `C++14/17`
* **Особенности реализации:** Объектно-ориентированная архитектура (ООП), безопасная работа с памятью, строгая обработка исключений при поврежденных бинарных файлах.
