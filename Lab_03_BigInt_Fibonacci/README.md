# Длинная арифметика: Фибоначчиева система счисления (C++)

Реализация структуры данных для работы с числами произвольной длины. В отличие от классической реализации в десятичной или двоичной системах, данный проект использует **фибоначчиеву систему счисления**, базирующуюся на теореме Цекендорфа.

## Особенности реализации

* Представление целых чисел осуществляется в виде суммы чисел Фибоначчи. Согласно теореме Цекендорфа, каноническое представление не содержит двух подряд идущих единиц (`11` запрещено).
* Динамическое выделение памяти под разряды реализовано на базе `std::vector<int>`, что исключает утечки памяти и позволяет числу расти неограниченно в зависимости от доступной ОЗУ.
* Класс `BigIntFibonacci` полностью интегрирован в семантику C++ через перегрузку арифметических (`+`, `-`, `*`, `%`) и логических операторов (`<`, `>`, `==`, `<=`, `>=`, `!=`).
* Встроенный алгоритм приведения произвольного массива битов к канонической форме (устранение ведущих нулей, обработка переполнений разрядов и запрещенных подстрок `11`).

## Технический стек

* Использование `explicit` конструкторов для защиты от неявного приведения типов (например, `std::string` -> `BigIntFibonacci`).
* Оптимизация локальности данных при арифметических сдвигах и делении.

## Примеры операций

Сложение и умножение выполняются поразрядно, после чего вызывается метод `normalize()`, восстанавливающий уникальное цекендорфово представление числа.
